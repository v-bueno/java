
# Projet Java Quizz

## Version de Java
Ce projet peut être exécuté avec Java 17 ou Java 20 et probablement d'autres versions.

## Instructions de Compilation
Pour compiler le projet, exécutez simplement la classe `Main`. Une fois que le main est lancé, il faut agrandir et réduire la fenêtre. Assurez-vous que vous disposez d'une installation de Java configurée correctement sur votre système.



## Prérequis
Avant de compiler et d'exécuter le projet, assurez-vous d'avoir téléchargé et ajouté la bibliothèque opencsv-3.8.jar à votre projet IntelliJ IDEA. Voici comment procéder :

1. Téléchargez opencsv-3.8.jar à partir du site officiel OpenCSV : [Télécharger opencsv-3.8.jar](https://sourceforge.net/projects/opencsv/files/opencsv/3.8/opencsv-3.8.jar/download)

2. Ouvrez votre projet dans IntelliJ IDEA.

3. Dans la barre de menu supérieure, cliquez sur "File" (Fichier) puis sélectionnez "Project Structure" (Structure du Projet).

4. Dans la fenêtre "Project Structure", sélectionnez "Libraries" (Bibliothèques) dans la colonne de gauche.

5. Cliquez sur le signe "+" en haut à gauche pour ajouter une nouvelle bibliothèque.

6. Sélectionnez "Java" dans le menu déroulant et recherchez le fichier opencsv-3.8.jar que vous avez téléchargé.

7. Cliquez sur "OK" pour ajouter la bibliothèque à votre projet.

Maintenant, la bibliothèque OpenCSV est ajoutée à votre projet IntelliJ IDEA et vous pouvez l'utiliser pour lire et écrire des fichiers CSV.

## Informations Utiles aux Tests
Pour les tests du projet, deux fichiers CSV sont nécessaires : `Parties.csv` et `Questions.csv`. Assurez-vous que ces fichiers sont présents dans le répertoire du projet et ne sont pas supprimés. Vous pouvez les recréer avec des en-têtes vides si nécessaire.

**Note importante :** Le fichier `Compte.data` n'est pas inclus dans le dépôt initial. Cependant, il sera automatiquement créé lors de la création de votre premier compte.

Veillez à prendre en compte ces informations lors de l'exécution des tests pour vous assurer que les fichiers requis sont présents et que le fichier `Compte.data` est généré correctement.
